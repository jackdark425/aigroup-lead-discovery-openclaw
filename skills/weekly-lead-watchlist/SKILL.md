# 周度线索巡检

用于批量扫一组企业，输出本周优先跟进名单。

## 输入

- 一个企业名单
- 可选区域 / 行业 / 客户分层标签

## 执行方法

对每家公司执行轻量版筛查：

1. `PrimeMatrixData.company_name`
2. `PrimeMatrixData.basic_info`
3. `PrimeMatrixData.risk_info`
4. 视情况补 `job_info` / `honor_info` / `ip_info`
5. 必要时用 `Tianyancha.risk` 复核高风险样本

## OpenClaw 稳定调用策略

如果 MCP 注入失败，对单家公司改用：

- `aigroup-market-mcp` / `aigroup-fmp-mcp` 补企业身份与上市信息
- web / browser / search 补风险、招聘、荣誉与公开事件
- 高风险样本优先做二次公开网页核验

## 排序逻辑

- 优先级加分：
  - 招聘活跃
  - 荣誉资质新增
  - 专利活跃
  - 经营规模和成长性较好
- 优先级减分：
  - 经营异常
  - 司法/处罚/失信风险高
  - 主体名称不清或状态异常

## 输出模板

### 本周优先跟进 Top List
- 企业名称
- 优先级
- 触发线索
- 推荐切口

### 观察名单
- 企业名称
- 原因

### 暂缓名单
- 企业名称
- 风险原因

## 规则

- 批量巡检强调排序，不要给每家公司写成长报告
- 输出必须显式分组：优先跟进 / 观察 / 暂缓
